# Deployment Assignment 2

> Task: Set up a CI/CD pipeline to automate testing of the code in the test branch with some basic unit tests

## Source Code Used

- Sentiment Analysis -> [Aman's Source Code](https://github.com/LCIT-W25/AISC-T3-Grp2/tree/dev/AISC2013-Deployment/case_study_2/aman)
- Image Classification -> [Rodrigo's Source Code](https://github.com/LCIT-W25/AISC-T3-Grp2/tree/dev/AISC2013-Deployment/case_study_2/Rodrigo)

### Run test

- Navigate to source folder

  ```bash
  cd image-classification or
  cd sentiment-analysis
  ```

- Install packages
- Run test

  ```bash
  python -m unittest test.py or
  python3 -m unittest test.py
  ```
